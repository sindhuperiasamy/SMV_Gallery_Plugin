== Description ==

A SMV Gallery Plugin is a simple plugin that uses the native WordPress media manager to add image to your site.

= A SMV Gallery Plugin Features =

* Responsive
* MU Compatible
* Handles image resizing and alignment
* Link the image
* Add title and description

= Quality You Can Trust =

SMV Gallery Plugin is developed and maintained by sindhu periasamy. The quality and bug is managed by the individual and plugin upgrading are in progress.

SMV Gallery Plugin features include:

* Add Image, gallery title, gallery excerpt, add gallery content and gallery image


== Installation ==

= Install =

Getting started with SMV Gallery Plugin is a breeze!

1. Download and install the SMV Gallery plugin
1. From your WordPress admin screen, select Plugins from the menu
1. Activate the SMV Gallery Plugin
1. Go to MSV Galleries in the Menu and Add galleries


= Requirements =

* PHP 5.2 or above
* WordPress 3.5 or above
